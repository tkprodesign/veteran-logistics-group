# Email Template Preview Folder

This folder contains browser-previewable HTML email templates using inline CSS for email-client compatibility.

Open `/email-templates/index.php` in your browser to review each template.

## Templates included
- signup-verification.html
- verification-code-resend.html
- forgot-password-reset.html
- shipment-status-sender.html
- shipment-status-receiver.html
- service-quote-internal-alert.html
- service-quote-customer-confirmation.html
- free-quote-admin-alert.html
- support-message.html
- newsletter-admin-alert.html
- newsletter-thank-you.html
